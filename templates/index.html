Attendance Guru

QR code based automated attendance system

Awesome Features of ATTENDANCE GURU App

  • A simple web app for students, teachers and parents to operate, organize and monitor the attendance.
  • Proxy proof due to time + student's unique id binded QR code. (i.e., for e.g. a certain QR code is valid for only 35 min.)
  • Unique QR code for every student for every lecture; sent to respective students via e-mail.
  • In-built QR code scanner in teacher dashboard for close monitoring of attendance marking procedure.
  • Facility to organize and maintain the attendance data in teacher's module.

Solves the following problems

  • Malpractises by students which generates untruthful attendance data which is unfair.
  • Attendance marking manually is tedious & long process (sometimes feels like infinity) for teachers & students.
  • As mentioned above it takes significant chunk of time from the allotted time for the lecture moreover with tampered data.
  • Data collection, organization, report generation is exhausting process and is prone to cause errors.
  • As this whole process involves multiple parties and the subprocesses work independently there is lack of synchronization and seamless flow of data.
